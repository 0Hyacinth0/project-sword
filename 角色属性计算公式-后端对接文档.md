## 一、公式参数常量

| 常量名 | 值 | 说明 |
|:---|:---:|:---|
| `STR_TO_HP` | 5 | 1 点力量 = 5 HP |
| `INT_TO_MP` | 5 | 1 点智力 = 5 MP |
| `STR_TO_PHYSICAL_ATTACK` | 2 | 1 点力量 = 2 物攻 |
| `INT_TO_MAGIC_ATTACK` | 2 | 1 点智力 = 2 魔攻 |
| `AGI_TO_PHYSICAL_ATTACK` | 0.5 | 1 点敏捷 = 0.5 物攻 |
| `AGI_TO_DODGE` | 0.005 | 1 点敏捷 = 0.5% 闪避率 |
| `AGI_TO_CRITICAL` | 0.004 | 1 点敏捷 = 0.4% 暴击率 |
| `BASE_HP` | 80 | HP 基础值 |
| `BASE_MP` | 20 | MP 基础值 |
| `BASE_DEFENSE` | 5 | 防御基础值 |

## 二、职业加成系数

不同职业对衍生属性有额外的百分比加成：

### 1. 战士 (profession = 1)

| 加成项 | 值 | 说明 |
|:---|:---:|:---|
| `hpMultiplier` | 1.2 | HP +20% |
| `mpMultiplier` | 0.8 | MP -20% |
| `physicalAttackMultiplier` | 1.15 | 物攻 +15% |
| `magicAttackMultiplier` | 0.8 | 魔攻 -20% |
| `defenseBonus` | 3 | 防御额外 +3 |

### 2. 法师 (profession = 2)

| 加成项 | 值 | 说明 |
|:---|:---:|:---|
| `hpMultiplier` | 0.8 | HP -20% |
| `mpMultiplier` | 1.3 | MP +30% |
| `physicalAttackMultiplier` | 0.7 | 物攻 -30% |
| `magicAttackMultiplier` | 1.2 | 魔攻 +20% |
| `defenseBonus` | 0 | 无额外防御 |

### 3. 猎人 (profession = 3)

| 加成项 | 值 | 说明 |
|:---|:---:|:---|
| `hpMultiplier` | 0.9 | HP -10% |
| `mpMultiplier` | 0.9 | MP -10% |
| `physicalAttackMultiplier` | 1.0 | 物攻无加成 |
| `magicAttackMultiplier` | 0.9 | 魔攻 -10% |
| `defenseBonus` | 1 | 防御额外 +1 |

## 三、计算公式（Java 实现）

```java
/**
 * 角色衍生属性计算
 * @param str 力量
 * @param int 智力
 * @param agi 敏捷
 * @param profession 职业编号：1-战士, 2-法师, 3-猎人
 * @return 衍生属性对象
 */
public DerivedStats calculateBaseStats(int str, int int, int agi, int profession) {
    // 获取职业加成配置
    ProfessionBonus bonus = getProfessionBonus(profession);
    
    // HP = (基础值 + 力量×5) × 职业倍率
    int maxHp = (int) Math.floor((BASE_HP + str * STR_TO_HP) * bonus.getHpMultiplier());
    
    // MP = (基础值 + 智力×5) × 职业倍率
    int maxMp = (int) Math.floor((BASE_MP + int * INT_TO_MP) * bonus.getMpMultiplier());
    
    // 物攻 = (力量×2 + 敏捷×0.5) × 职业倍率
    int physicalAttack = (int) Math.floor(
        (str * STR_TO_PHYSICAL_ATTACK + agi * AGI_TO_PHYSICAL_ATTACK) 
        * bonus.getPhysicalAttackMultiplier()
    );
    
    // 魔攻 = 智力×2 × 职业倍率
    int magicAttack = (int) Math.floor(
        (int * INT_TO_MAGIC_ATTACK) * bonus.getMagicAttackMultiplier()
    );
    
    // 防御 = 基础值 + 职业额外加成
    int defense = BASE_DEFENSE + bonus.getDefenseBonus();
    
    // 闪避率 = 敏捷×0.005（百分比）
    double dodgeRate = agi * AGI_TO_DODGE;
    
    // 暴击率 = 敏捷×0.004（百分比）
    double criticalRate = agi * AGI_TO_CRITICAL;
    
    return new DerivedStats(maxHp, maxMp, physicalAttack, magicAttack, defense, dodgeRate, criticalRate);
}

/**
 * 职业加成配置
 */
private ProfessionBonus getProfessionBonus(int profession) {
    switch (profession) {
        case 1: // 战士
            return new ProfessionBonus(1.2, 0.8, 1.15, 0.8, 3);
        case 2: // 法师
            return new ProfessionBonus(0.8, 1.3, 0.7, 1.2, 0);
        case 3: // 猎人
            return new ProfessionBonus(0.9, 0.9, 1.0, 0.9, 1);
        default:
            return new ProfessionBonus(1.0, 1.0, 1.0, 1.0, 0);
    }
}

@Data
@AllArgsConstructor
public class ProfessionBonus {
    private double hpMultiplier;
    private double mpMultiplier;
    private double physicalAttackMultiplier;
    private double magicAttackMultiplier;
    private int defenseBonus;
}

@Data
@AllArgsConstructor
public class DerivedStats {
    private int maxHp;
    private int maxMp;
    private int physicalAttack;
    private int magicAttack;
    private int defense;
    private double dodgeRate;
    private double criticalRate;
}
```

## 四、应用场景

### 1. 创建角色时

初始化属性后，调用计算公式生成衍生属性：

```java
// 根据职业获取初始属性
int[] baseAttrs = getBaseAttributes(profession); // [str, int, agi]
int str = baseAttrs[0];
int int = baseAttrs[1];
int agi = baseAttrs[2];

// 计算衍生属性
DerivedStats derived = calculateBaseStats(str, int, agi, profession);

// 写入数据库
character.setStrength(str);
character.setIntelligence(int);
character.setAgility(agi);
character.setMaxHp(derived.getMaxHp());
character.setHp(derived.getMaxHp());
character.setMaxMp(derived.getMaxMp());
character.setMp(derived.getMaxMp());
character.setPhysicalAttack(derived.getPhysicalAttack());
character.setMagicAttack(derived.getMagicAttack());
character.setDefense(derived.getDefense());
character.setDodgeRate(derived.getDodgeRate());
character.setCriticalRate(derived.getCriticalRate());
```

### 2. 属性加点时

加点后重新计算衍生属性：

```java
// 校验加点总数
int total = strAdd + intAdd + agiAdd;
if (total > character.getAvailablePoints()) {
    return error("可用属性点不足");
}

// 更新基础属性
character.setStrength(character.getStrength() + strAdd);
character.setIntelligence(character.getIntelligence() + intAdd);
character.setAgility(character.getAgility() + agiAdd);
character.setAvailablePoints(character.getAvailablePoints() - total);

// 重新计算衍生属性
DerivedStats derived = calculateBaseStats(
    character.getStrength(),
    character.getIntelligence(),
    character.getAgility(),
    character.getProfession()
);

// 更新衍生属性
character.setMaxHp(derived.getMaxHp());
character.setMaxMp(derived.getMaxMp());
character.setPhysicalAttack(derived.getPhysicalAttack());
character.setMagicAttack(derived.getMagicAttack());
character.setDefense(derived.getDefense());
character.setDodgeRate(derived.getDodgeRate());
character.setCriticalRate(derived.getCriticalRate());

// 注意：当前 HP/MP 不自动调整，允许超过上限或保持原值
```

## 五、完整属性汇总（含装备和战宠）

角色的最终属性 = 基础属性 + 装备加成 + 出战战宠加成

```java
public CharacterStats calculateFullStats(Character character) {
    // 基础衍生属性
    DerivedStats base = calculateBaseStats(
        character.getStrength(),
        character.getIntelligence(),
        character.getAgility(),
        character.getProfession()
    );
    
    // 装备加成汇总
    DerivedStats equip = sumEquipmentStats(character.getEquipment());
    
    // 战宠加成（仅出战战宠）
    PetBonus pet = getActivePetBonus(character.getActivePet());
    
    // 最终属性
    return new CharacterStats(
        base.maxHp + equip.maxHp + (pet.hp != null ? pet.hp : 0),
        base.maxMp + equip.maxMp,
        base.physicalAttack + equip.physicalAttack + (pet.attack != null ? pet.attack : 0),
        base.magicAttack + equip.magicAttack,
        base.defense + equip.defense + (pet.defense != null ? pet.defense : 0),
        base.dodgeRate + equip.dodgeRate + (pet.dodgeRate != null ? pet.dodgeRate : 0),
        base.criticalRate + equip.criticalRate + (pet.criticalRate != null ? pet.criticalRate : 0)
    );
}
```

## 六、装备加成汇总

遍历六槽位装备，汇总属性加成：

```java
public DerivedStats sumEquipmentStats(EquipmentSlots slots) {
    int hp = 0, mp = 0, pa = 0, ma = 0, def = 0;
    double dodge = 0, crit = 0;
    
    for (Equipment eq : slots.getAll()) {
        if (eq == null) continue;
        EquipmentStats stats = eq.getStats();
        hp += stats.getHp() != null ? stats.getHp() : 0;
        mp += stats.getMp() != null ? stats.getMp() : 0;
        pa += stats.getPhysicalAttack() != null ? stats.getPhysicalAttack() : 0;
        ma += stats.getMagicAttack() != null ? stats.getMagicAttack() : 0;
        def += stats.getDefense() != null ? stats.getDefense() : 0;
        dodge += stats.getDodgeRate() != null ? stats.getDodgeRate() : 0;
        crit += stats.getCriticalRate() != null ? stats.getCriticalRate() : 0;
    }
    
    return new DerivedStats(hp, mp, pa, ma, def, dodge, crit);
}
```

## 七、战宠加成

出战战宠给主人提供的属性加成（参考战宠系统文档）：

```java
public PetBonus getActivePetBonus(Pet pet) {
    if (pet == null || !pet.getIsActive()) {
        return new PetBonus(0, 0, 0, 0, 0);
    }
    
    // 加成值 = 战宠属性 × 10% × 品质倍率
    double rarityMultiplier = getRarityMultiplier(pet.getRarity());
    
    int hpBonus = (int) Math.floor(pet.getHp() * 0.1 * rarityMultiplier);
    int attackBonus = (int) Math.floor(pet.getAttack() * 0.1 * rarityMultiplier);
    int defenseBonus = (int) Math.floor(pet.getDefense() * 0.1 * rarityMultiplier);
    
    return new PetBonus(hpBonus, attackBonus, defenseBonus, 0, 0);
}
```

## 八、注意事项

1. **精度处理**：HP/MP/攻击/防御使用 `Math.floor()` 取整，闪避/暴击保留 4 位小数
2. **前后端一致性**：确保公式参数、职业加成系数完全一致，否则会导致前后端数据不一致
3. **加点校验**：后端必须校验加点总数 ≤ `availablePoints`，不能仅依赖前端校验
4. **HP/MP 当前值**：加点后不自动调整当前 HP/MP，允许超过新上限（战斗后再修正）
5. **装备加成**：装备的 `dodgeRate` 和 `criticalRate` 是百分比数值（如 5 表示 5%），需注意与属性计算的百分比格式统一
